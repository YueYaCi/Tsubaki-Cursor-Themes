# 投稿指南

## 一、原创主题投稿（鼓励）

如果您是作者本人，投稿即视为同意以下条款：
- 您确认对该主题拥有完整版权
- 您同意本仓库以非独占方式展示、存储和分发该主题
- 建议您在 theme.json 中标注授权方式（如"禁止二创""允许非商用"等）

## 二、搬运/收集投稿

- 仅接受原作者明确允许再分发的资源
- 必须提供原出处链接
- 禁止搬运付费、加密或声明"禁止转载"的资源

## 三、文件命名规范
请确保光标文件按以下命名，否则软件无法识别：

| 角色 | 文件名 |
|------|--------|
| 正常选择 | Arrow.cur / Normal.ani / Pointer.ani |
| 帮助 | Help.ani |
| 后台运行 | Working.ani / AppStarting.ani |
| 忙 | Busy.ani / Wait.ani |
| 文本选择 | Text.ani / IBeam.cur |
| ... | ... |

## 必须包含
- `theme.json`（参考已有主题格式）
- 至少包含 Arrow 光标
