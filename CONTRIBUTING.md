# 投稿指南

## 文件命名规范
请确保你的光标文件按以下命名，否则软件无法识别：

| 角色 | 文件名 |
|------|--------|
| 正常选择 | Arrow.cur / Normal.ani / Pointer.ani |
| 帮助 | Help.ani |
| 后台运行 | Working.ani / AppStarting.ani |
| 忙 | Busy.ani / Wait.ani |
| 文本选择 | Text.ani / IBeam.cur |
| ... | ... |

## 必须包含
- `theme.json`（参考已有主题格式）
- 至少包含 Arrow 光标
